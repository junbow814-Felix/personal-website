# 王均博 Felix 个人网站

这是一个 Apple 风格的中文个人品牌单页网站首版，定位为市场营销 / 内容运营方向的作品展示与求职入口。顶部 Tab 作为页面切换入口；概览页包含首屏照片展示和发展时间线，其他 Tab 切换到对应内容页。

## 预览

建议用本地静态服务器运行当前目录，然后打开：

- 网站首页：`http://127.0.0.1:4175/index.html`
- 内容编辑器：`http://127.0.0.1:4175/editor.html`

编辑器可修改首页文案、时间轴、能力区、经历卡片、联系方式。页面边距和字号保持固定设计，不在编辑器中开放。

## GitHub Pages 部署

这是纯静态网站，不需要构建步骤。推送到 GitHub 后，可在仓库的 Settings → Pages 中选择：

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/root`

保存后等待 GitHub Pages 发布即可。

## 后续替换内容

- 常用文案：优先修改 `site-content.json`
- 可视化编辑入口：打开 `editor.html` 修改后，使用“保存文案到网站文件”
- 图片素材：替换 `assets/` 下对应文件

如果浏览器不支持直接写入文件，编辑器会自动改为下载新版配置文件，再手动替换同名文件即可。
